# Buy-and-Sell

A cryptocurrency buy and sell platform with user flows, admin management, and a modern UI.

## Project Structure

- `app/` - Frontend application (Vite + React + TypeScript)
- `backend/` - Backend services/APIs
- `fullstack-app/` - Additional fullstack implementation assets
- `images/` - Branding and asset files

## Frontend Quick Start

```bash
cd app
npm install
npm run dev
```

## Frontend Build

```bash
cd app
npm run build
npm run preview
```

## Deployment

The frontend is deployed on Vercel.
